Game Phases

Quest leader will propose a team

Leader Phase
The leader-torch passes over to the next player. Size of the team is determined by mission # and total number of players (will be shown on screen). 

All players will vote on the team proposal

Voting Phase
If majority accepts, move onto the Quest Phase. If majority rejects or there is a tie, go back to the Leader Phase (new leader). Mission will fail if 5 teams are rejected.               

Team will go on a Quest

Quest Phase
Blue team can only assist. Red team can either assist or sabotage the quest. One sabotage needed to fail a quest, except for Quest #4 when playing with 7-10 players

End of Game

3/5

Majority of 5 Quests
If Red team wins, Game Over. If Blue team wins, move onto Assassin phase.

Find and kill Merlin

Assassin
If successful, Red wins, else Blue wins.
